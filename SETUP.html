Choose Your Setup Method:

Option 1: Local by Flywheel EASIEST

Best for beginners and quick setup

  • One-click WordPress installation
  • Built-in MySQL and PHP
  • Beautiful user interface
  • Free forever
Download Local (Free)

Quick Steps:

  1. Install Local by Flywheel
  2. Create new site: "Inner Flow"
  3. Copy wp-content/themes/inner-flow to the site's themes folder
  4. Copy wp-content/plugins/inner-flow-events to the plugins folder
  5. Activate theme and plugin in WordPress admin

Option 2: XAMPP POPULAR

Full-featured Apache + MySQL + PHP stack

  • Complete development environment
  • phpMyAdmin included
  • Good for multiple projects
  • Free and open source
Download XAMPP

Quick Steps:

  1. Install XAMPP
  2. Move this folder to C:\xampp\htdocs\
  3. Start Apache and MySQL in XAMPP Control Panel
  4. Visit http://localhost/inner-flow
  5. Complete WordPress installation

Option 3: Docker FOR DEVELOPERS

Containerized environment (requires Docker)

  • Already configured with docker-compose.yml
  • Isolated environment
  • Easy to deploy
Download Docker

Quick Steps:

  1. Install Docker Desktop
  2. Open PowerShell in this folder
  3. Run:
    docker-compose up -d
  4. Visit http://localhost:8080
  5. Complete WordPress installation
